# Smart Service Booking System

## How to Run
1. Download the ZIP
2. Open index.html in a browser
3. Must be served locally (use VS Code Live Server) 
   because fetch() won't work by just double clicking the file

## Features
- Dynamic city, service and provider loading from JSON
- Async booking flow with 30% random failure simulation
- No page reloads

## Files
- index.html — structure
- style.css — styling
- script.js — logic
- data.json — data
